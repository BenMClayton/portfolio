# Deployment

## GitHub

Upload the project files to the root of:

`BenMClayton/portfolio`

Use the `main` branch.

## Render

The included `render.yaml` can create the site automatically. For a manual Static Site setup, use:

- Branch: `main`
- Build command: `npm run build`
- Publish directory: `dist`

After a commit is pushed to `main`, Render should build and deploy the updated site automatically.
